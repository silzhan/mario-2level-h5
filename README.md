# Super Mario World 2 — HTML5 Edition 🎮

A **Super Mario World 2** (World 2) fan game built from scratch in **pure JavaScript & HTML5 Canvas**. No frameworks, no libraries — just vanilla JS pixel art.

## 🕹️ Play Online

Just open `index.html` in any modern browser. No server required.

## 🎮 Controls

| Action       | Keys                          |
|-------------|-------------------------------|
| Move Left   | `←` / `A`                    |
| Move Right  | `→` / `D`                    |
| Duck        | `↓` / `S`                    |
| Jump / Swim | `Space` / `↑` / `W`          |
| Fireball    | `Shift` / `X`                |
| Restart     | `R`                          |
| Mute        | `M`                          |
| Pause       | `P`                          |
| Quit        | `Esc`                        |

> On the title screen, press `Enter` to start World 2-1, or `1`–`4` to pick a level directly.

## 🧱 Features

- **4 unique levels** with custom tile maps
- **Mario mechanics**: running, jumping (variable height), ducking, growing big, fire flower, star power
- **Enemies**: Goombas, Green & Red Koopas (shell kicking), Paratroopas, Blooper (squid), Cheep-Cheep, Piranha Plants, Hammer Bros, Podoboo, Bowser
- **Physics**: gravity, friction, swimming (buoyancy), lift platforms
- **Items**: Coins, Super Mushrooms, Fire Flowers, Star, 1‑Up Mushrooms
- **Block types**: Brick (breakable), Question Block, Hidden Block, Multi-coin Block
- **Interactive elements**: Pipes, flagpole, bridge collapse, axe grab
- **Score system**, time limit, lives, fireworks on clear
- **Pixel‑art rendering** with sprite sheets and hand‑drawn tile colors
- **Background music & SFX** (Web Audio API synthesis)
- **Title screen** with level selection

## 🛠️ Development

All game logic is in the `js/` folder. Levels are procedurally generated by `level.js` using 2D tile arrays. The rendering uses a sprite sheet (`img/sprites.png`) combined with programmatic shapes for tiles and entities.

To extend the game:

1. Add a new enemy class in `js/` and include the script in `index.html`.
2. Create a level map function in `level.js` and add it to `generateWorld2Map()`.
3. Register spawn calls and collision logic in `game.js`.
4. Draw the entity in `renderer.js`.
